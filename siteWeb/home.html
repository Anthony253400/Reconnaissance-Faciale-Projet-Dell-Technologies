About the project

This application was built as part of a curricular internship at Dell Technologies Montpellier. The goal is to explore real time face recognition.

The system detects faces and bodies through a webcam feed, matches them against a database of known individuals, and can identify people even when their face is not visible using body shape, height and appearance as signals.

How does it work?

01

Face detection

Detects faces in real time using MediaPipe BlazeFace and generates a unique 512-dimensional embedding with ArcFace for each person.

02

Body re-identification

Recognises people from behind or at a distance using body shape and appearance, even without a visible face, through a YOLOv8 detection pipeline.

03

Vector database

Embeddings are stored in Qdrant and matched using cosine similarity for fast and accurate recognition in real time.

04

GDPR compliant

No photos are stored only biometric vectors. Explicit consent is required before any data is saved to the database.
