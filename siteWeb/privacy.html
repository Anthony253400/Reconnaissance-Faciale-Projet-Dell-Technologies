Politique de confidentialité

Qui sommes-nous ?

Cette application a été développée par Anthony Miranda et Lea Carminati dans le cadre d'un projet de stage curriculaire de reconnaissance faciale chez Dell Technologies Montpellier.

Quelles données collectons-nous ?

Nous collectons les données suivantes :

  • Nom et prénom
  • Données biométriques du visage issues de votre photo

Pourquoi collectons-nous ces données ?

Ces données sont collectées dans le but de développer une application de reconnaissance faciale dans le cadre de ce projet de stage.

Comment sont stockées vos données ?

Les photos envoyées sur le site ne sont pas stockées dans notre base de données. Seuls les vecteurs biométriques issus du traitement de votre photo sont conservés. Ces données seront conservées pendant la durée du projet.

Qui a accès à vos données ?

Seuls les réalisateurs de ce projet (Anthony Miranda et Lea Carminati) ont accès à vos données biométriques. Aucune donnée n'est partagée avec des tiers.

Consentement

Conformément au RGPD, le traitement de vos données biométriques nécessite votre consentement explicite. Vous pouvez le fournir en cochant la case "I agree to the privacy policy" sur notre site.

Vos droits

Vous avez le droit de demander la suppression de vos données de notre base de données à tout moment. Pour exercer vos droits, vous pouvez nous contacter aux adresses mail suivantes :

  • Anthony Miranda : anthony.miranda@etu.univ-mtp3.fr
  • Lea Carminati : lea.carminati@etu.univ-mtp3.fr
